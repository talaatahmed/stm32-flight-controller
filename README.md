# STM32 Flight Controller

A modular STM32-based flight controller evolving from simple polling to FIFO architecture.

## Structure
- FC_STM32_v1_Baseline
- FC_STM32_v2_FIFO_Frontend

## Versions
### v1
- Polling IMU
- Basic PID

### v2
- FIFO (1kHz)
- Better architecture

## Next
- EXTI
- DMA
- RTOS
